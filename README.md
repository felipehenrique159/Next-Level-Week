## Sobre a aplicação

Desenvolvimento back-End da aplicação da semana Next-Level-Week chamado Ecoleta, aplicação construída utilizando 
NODE JS como back end, a aplicação consiste em cadastro de pontos de coletas de resíduos!

Aplicação simples e intuitiva que usa conceitos bem legais do node js

## Tecnologias utilizadas:

- ### Express
- ### Cors
- ### Knex
- ### SqLite3
- ### TypeScript
